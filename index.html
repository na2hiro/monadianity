モナド教 - Monadianity

前提知識：モナド

モナドを理解せずともモナド教を信ずることは出来ますが，理解していればより深く納得できるでしょう．

  • 操作 :: 型 -> 型 は，"型"から"型"へ写す"操作"の存在を表します．
  • モナドの文脈 m が必要とする2つの操作：
    • return :: a -> m a で，値を保ちつつ文脈 m の中に入れ込むことが出来ます．
    • (=<<) :: (a -> m b) -> (m a -> m b) で，「値を文脈に入った別の値へ写す操作」を「文脈に入った値を同じ文脈に入った別の値へ写す操作」に変換します．
  • id :: a -> a は値をそのまま返す操作です．
  • id を =<< で変換して得られる操作 join :: m (m a) -> m a で，二重に文脈に入った値を一重の文脈に入った値に戻すことが出来ます．
  • 文脈の値から生の値を取り出す型 m a -> a を持つ操作は，一般に用意されていません．

教義

死をモナドの文脈とみなし， a を現世， m a をあの世の人であると仮定することで，以下のことを導きます．

  • return :: a -> m a が死ぬことに対応します．
  • あの世からこの世に戻ってくる( m a -> a 型の)操作は用意されていません．つまり死んだら現世に生き返ることは出来ません．これは私たちも知っている事実ですね．
  • join :: m (m a) -> m a は，あの世のあの世からあの世に生き返ることに対応します．
    • つまりあの世にもあの世があるという主張が導かれました．
    • あの世で死んでも生き返ることができるのです．

Q&A

天国なんて確認できないし，あるわけがない．
確かに現世の我々は天国の有無を確認することができないのですが，ないことを証明するのは困難です．なお，キリスト教，イスラム教，仏教，神道は天国の存在を認めており，デファクトスタンダードになっています．
天国ではみんな幸せなの？
はい．天国では，死んでも生き返れます．つまり，人を殺しても咎められることはないし，自分の死を恐れること無く何にでも挑戦できます．このことをもって，あの世ではみな幸せに暮らしているのです．
批判するつもりはありませんが，他の宗教では，天国に行くとあらゆる快楽が得られる(例えば酒が飲み放題だ)とか，また一方で，煩悩を超越し永遠の安らぎを得られるともいいます．しかし，酒はどこから湧いて出るのか，あなたは本当に安らいでいるだけでいいのか，考えてみてください．天国に行っても人間は人間，というほうがよりもっともらしいのではないでしょうか．
死ぬと天国に行くのなら，天国は無限の人で溢れかえるのでは？
いいえ．天国の天国で死んだら，天国の天国の天国に行きます．つまり，天国は無限に連なっているのです．天国は無限の死人を受け入れることができるのです．
この点で，既存のどの天国よりももっともらしいと言えます．
天国はどうなっているの？
行ったことがある人がこの世の中にはいないのでわかりませんが，あらゆる値と操作はそれぞれ return :: a -> m a と fmap :: (a -> b) -> (m a -> m b) と自然性条件によって天国でも通用することが担保されているので，現世の物理法則などもそのまま存在しているはずです．現世と似たような世界になっていると推測できます．
死んだ後，自我とか知識は次の世界でも保たれるの？
return :: a -> m a は a つまり人を何も変えずに文脈に入れる操作であり，通常は保たれます．つまり，知識をつけることはあらゆる世界において善いことです．
ただし， data Heaven a = H のように天国モナドが幽霊型である場合はその限りではありません．この例だと，全て生物は死後唯一の存在へと収束してしまいます．
では同様に，身体的特徴は次の世界でも保たれるの？
わかりません．そもそも生きているとは意識を持っていることだと考えられます．例えば髪の毛を切ったり，腕を失ったりしても相変わらず自分は生きています．将来脳みそ以外人工物で出来たアンドロイドが実現したとしても，意識があれば生きているとみなせるでしょう． return が意識をそのまま保って天国に移すものだとしても，身体的特徴は直接生死とは関係なく，保たれるかどうかはわかりません．また，もし保たれると仮定すると，寝たきりの老人が死んだら天国でも寝たきりということになり，少し残酷です．
なぜ「死ぬこと」が return と名付けられているの？
return to the soil(土に還る)に由来します．
輪廻転生とはどう違うの？
輪廻転生では，死んだ後に行く世界が現世であり，一見よく似ています．しかし，生き返る操作の存在を許している点が大きく違います．(関手に関する表現に誤りがあり訂正しました)
教典はありますか？
モナドがモナドたる所以としての return と =<< の概念と，自ずから導かれる join の概念，そして死をモナドの文脈に対応させること自体が教典であり，Q&Aも含めたその他の文書はモナド教における単なる解釈の一つです．
要は困ったら死ねばいいってこと?
もしかすると死に相当するモナド m が唯一ではないかもしれません．例えば死後の世界は天国と地獄の2つがあるという宗教もあります．先ほどのように幽霊型であるモナドかもしれません．果たしてどこに行くかは死んでみないとわかりませんし，一度そのモナドに入ったが最後，抜け出して別のモナドに入り直すことは永遠に出来ません．現世にいる限り，気軽に死なないほうが無難でしょう．天国に行ってからは思う存分にどうぞ．
哲学のモナドは関係ありますか？
ありません．
モナド教での天国のモデルよりもっともらしいモデルもあるのでは？
モナド教は，既存の天国の概念よりもっともらしいモデルを提唱するもので，唯一完全な天国を提唱するものではありません．是非あなたがよりもっともらしいモデルを提案してください．

寄せられたQ&A

もしも return が自明な実装でない場合、人としての本質が変容する、つまり型が同じでも値は異なるという可能性がないですか？(質問)
モナド則のうちの以下の2つにより，元の本質は取り出せる形で保たれます．
  • f =<< (return x) == f x
  • return =<< m == m
join で生き返った人にも同様に，他の人になるのではないでしょうか。 id を用いるもの以外にも定義できる場合があるので。(質問)
id 以外の操作を =<< に渡した場合，確かに他の人や他の生物になる可能性はあります．ただ， id (私が私になるという自明の理)以外の操作の存在は確認していません．
モナド教においては天国は多重だそうですが、もしかして我々が生きている現世も、どこかの世界からみた天国なのですか？それとも現世は大量に連なる天国の鎖の中の端点なんですか？(質問)
一度死んで天国を見てきたものがこの世に向かって"生き返る"ことがあれば，無数に連なる天国の途中であると確認できますが，これまでにそのような人がいたことがないため，端点にあるものと判断できます．
m a -> a の存在を追い求める人たちは邪教になるのですか？(質問)
もしこの世への生き還りが実現したとすれば，我々の世界は既にどこかの世界の天国であった，つまりただの m (m a) -> m a であっただけのことで，依然としてモナド教の範疇です．
